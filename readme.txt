== Description ==
This plugin shows the latest posts with thumbnail. This has the full control of the content and easily customizable. Widget and shortcode ready.


== Installation ==
* Upload `SB Latest Posts` to the `/wp-content/plugins/` directory of your application
* Login as Admin
* Activate the plugin through the 'Plugins' menu in WordPress


= Usage shortcode for any page =
`[spellbit_latest_posts img="yes" date="yes" tag="yes" cat="yes" words="30"]

For any template
<?php echo do_shortcode('[spellbit_latest_posts img="yes" date="yes" tag="yes" cat="yes" words="30"]'); ?>`
